# Run and deploy your app

This repository contains everything you need to run the app locally.

View the original AI Studio project:
https://ai.studio/apps/0f752010-36ab-4176-8b2e-54f0224d5986

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `VITE_GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`
